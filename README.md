# Pointer

Conceito de ponteiro, com passagem de referência ou por cópia em C++

### Compilar

`g++ -o ponteiro ponteiro.cc`

### Executar

`./ponteiro`

### Debug
Criei uma função debug que imprime: "[__LINE__] __FUNCTION__(value);"
